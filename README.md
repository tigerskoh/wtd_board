# 실행순서
0. mysql 다음과 같이 셋팅 합니다.
```
	host: '127.0.0.1',
	user: 'root',
	port: 3306,
	// 패스워드 없음
```
1. 패키지 root 에서 npm install
2. npm start 로 서버실행
3. npm test 로 통합테스트
